Welcome to GraphVisualiser

This is the Project Homepage for Ontology Visualiser Project assigned as a part of Structured System Analysis and Design course.
The graph is rendered by a force based Algorithm, which models the nodes as charged particles and edges as springs.

The tool can be downloaded as a tarball or zipball simply by clicking above links. Or incase you love black terminals with green fonts or are too lazy to unzip or untar tarballs(as are we), Simply fire up your terminal and run 

$ git clone http://github.com/piyushbansal/GraphVisualiser

System Requirements

A Modern Browser.
Yes, That should be about it, I guess. Apart from that , You would need to have php along with a web server installed on your local machine.

Technology And Algorithms

The tool uses Javascript , and has been written in d3 library of javascript. It uses lightweight JSON as a document-oriented database.Since graphs have some sort of structuring , we thought that JSON would be our best bet,since its fast,efficient and easy to manage. Server end scripting has been done in PHP. Dependencies have been kept as low as possible and the tool doesn't depend on any other external module. The force based algorithm used is inspired by Tim Dwyer and Thomas Jakobsen
